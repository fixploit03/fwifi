## WI-FI HACKING

## Peralatan

Berikut ini adalah alat-alat yang diperlukan untuk melakukan peretasan WI-FI.

- Sistem operasi Linux (direkomendasikan Kali Linux)
- WI-FI Adapter (yang support monitor mode & frame injection)

## Instal Tools Aircrack-ng

```
$ sudo apt-get update

$ sudo apt-get upgrade

$ sudo apt-get install git

$ git clone https://github.com/rofidoang03/wifi-hacking/

$ cd wifi-hacking/aircrack-ng

$ chmod +x instal.sh

$ ./instal.sh
```
